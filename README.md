# Ola-Ride-Booking-Analysis
Power BI dashboard & SQL analysis of 100K+ Ola ride bookings — insights on revenue, cancellations, and customer ratings.

- Power BI dashboard analyzing 100,000+ Ola ride bookings from July 2024
- Covers booking success/cancellation rates, revenue by vehicle type and payment method, and driver/customer ratings
- Built across 5 interactive report pages
