# Ola-Ride-Booking-Analysis
Power BI dashboard & SQL analysis of 100K+ Ola ride bookings — insights on revenue, cancellations, and customer ratings.

**Project Objective**
To analyze booking performance, revenue, cancellation patterns, and service ratings of Ola ride bookings over a full month using an interactive Power BI dashboard.

**Key Business Questions**
What is the overall booking success and cancellation rate?
Which vehicle types generate the highest booking value and distance travelled?
Which payment methods drive the most revenue?
What are the leading causes of ride cancellations, from both the customer's and driver's side?
How do driver and customer ratings vary across vehicle types?
Who are the top customers by booking value?

**Process**
Data Cleaning — Cleaned and standardized raw booking data
DAX Measures — Created calculated measures for KPIs including total bookings and cancellation rate
Dashboard Design — Built a 5-page interactive dashboard with custom page navigation, slicers, and visualizations across Overall, Vehicle Type, Revenue, Cancellation, and Ratings views
